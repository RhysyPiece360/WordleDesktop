## How to compile (for you developer nerds)
<br/>
Clone the repository.<br/>
Install/Compile GTK3 and CMake.<br/>
`cd WordleDesktop`<br/>
`cmake .`<br/>
`cmake --build .`<br/>
Linux/Mac:<br/>
`sudo cmake --install .`<br/>
Windows:<br/>
In an admin terminal `cmake --install .`<br/>
